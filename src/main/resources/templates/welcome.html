About Kiosk Task Manager

Kiosk Task Manager is a simple task management application designed for small businesses. It allows you to easily assign, track, and manage tasks using your smartphone.

Key Features

  • Simple user hierarchy
  • Flexible deadline and notification customization
  • Automatic translation of tasks
  • 10 basic scenarios for a quick start

Getting Started

Download the Android application, create your user hierarchy, add tasks, and start managing them.
